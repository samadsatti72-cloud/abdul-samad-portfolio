# Abdul Samad Satti Portfolio

A professional AI/ML Engineer portfolio website with a modern dark neon UI, animated background network graph, typing intro effect, project showcase, and contact form.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Vercel (deployment)

## Sections
- Hero
- About
- Skills + proficiency bars
- Featured Projects
- Experience
- Education
- Contact + Formspree form

## Local Development
Open `index.html` directly in browser, or serve with a static server:

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Contact Form Setup
The site currently uses Formspree endpoint:

`https://formspree.io/f/xblyznqr`

If you want to use your own form endpoint:
1. Create a form on Formspree.
2. Replace the `action` in `index.html` contact form.

## Deploy on Vercel
1. Push repository changes to GitHub.
2. Import repo in Vercel.
3. Framework preset: `Other`.
4. Build command: none.
5. Output directory: root.
6. Deploy.
